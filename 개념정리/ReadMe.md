# 0619 과제

### 클래스 (Class) : "설계도" 또는 "틀"

- 특정한 객체를 만들어내기 위해 변수(상태)와 메소드(동작)를 미리 정의해 놓은것.
    - 형태와 기능이 어떻게 될지 규격만 정해둔 것이지, 그 자체로 동작하지 않음.
    - 클래스 자체는 (정적 영역을 제외하면) 메모리 공간을 차지하거나 실제 데이터를 담고 있지 않음.

### 객체 (Object / Instance) : "실체"

- 클래스를 바탕으로 컴퓨터 메모리 공간(Heap)에 실제로 할당되어 생성된 **실체.**
    - 자바에서는 `new` 키워드를 사용해서 클래스로부터 객체를 생성함.
    - 하나의 클래스로 여러 개의 객체를 만들 수 있고, 생성된 객체들은 모양(구조)은 같아도 서로 다른 고유한 데이터를 가질 수 있음.

해당 파일은
 간단하게 카페명과 지점명 메뉴 이름과 가격등을 생성하는 파일들임.
